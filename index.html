Coding Quiz Challenge

Welcome to the Coding Quiz Challenge! In this quiz, you will be asked a scope of questions about HTML, CSS, and JavaScript. This quiz is a timed quiz. Once the quiz begins, the timer will start. Any questions answered incorrectly will subtract time from the timer and you only get once chance to answer each question, so be sure to try your best at answering the question the first time! The quiz ends when you are finished or the timer reaches zero, then you will be given a score. Feel free to use any resources necessary to complete the quiz. Good luck!
